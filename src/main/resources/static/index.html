🟢 Импорт участников из Google Sheets

📥 Одноразовая миграция — импорт всех существующих участников

Читает все строки из Google Sheets и сохраняет участников в базу данных PostgreSQL.
Дубликаты по email/iin пропускаются автоматически. Статус определяется по цвету строки в таблице.
🟢 Зелёный → ACTIVE  |  🟣 Фиолетовый → ECO_YOUTH  |  🔵 Синий → BOARD_MEMBER  |  🔴 Красный → LEFT  |  ⬜ Белый → REMOTE

Нажмите кнопку чтобы начать импорт...
🔗 Webhook — автоматическое добавление новых участников

Новые записи из Google Forms автоматически попадают в БД через Google Apps Script.
Убедитесь что в Apps Script триггер onFormSubmit настроен и URL сервера прописан верно.

POST http://localhost:8080/api/webhook/sheets

Эндпоинты, для которых нет отдельной вкладки в UI. Публичные вызовы работают без токена; остальные — после входа (HR для корректировки часов).

—

GET /api/events/{id}/attendees

—

POST /api/volunteers/{id}/hours/adjust

POST /api/webhook/sheets

Публичный webhook (без JWT). JSON — SheetsWebhookDto.

—

POST /api/v1/public/registration/google-sheets

Требует JWT (не в permitAll). JSON — GoogleSheetRegistrationDto.
